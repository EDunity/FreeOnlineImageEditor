使い方

このツールの使い方です。

ホイールドラッグ : 画像を移動
Ctrl + S : 画像をエクスポート
Ctrl + Z : 取り消し
Ctrl + Y : やり直し
  • Pencil
    • 右クリック : 色を抽出
    左クリック : 点を配置
  • Line
    • 右クリック : 色を抽出
    左クリック : 直線を配置
  • Rect
    • 右クリック : 色を抽出
    左クリック : 四角形を配置
  • Circle
    • 右クリック : 色を抽出
    左クリック : 円を配置
  • Brush
    • 右クリック : 色を抽出
    左クリック : ブラシ
  • Bucket
    • 右クリック : 色を抽出
    左クリック : 塗りつぶし
  • Select
    • 右クリック : 範囲を移動
    左クリック : 範囲を選択
    Ctrl + A : 全範囲を選択
    Ctrl + G : 選択した範囲を塗りつぶし
    Ctrl + C : 選択した範囲をコピー
    Ctrl + V : コピーした範囲をペースト
    Ctrl + D : コピーした範囲を元に画像を作成

注意事項

  • PNG、JPG、JPEG、WEBP以外の拡張子をインポートすることはできません。
  • PNG、JPG、JPEG、WEBP以外の拡張子をエクスポートすることはできません。
  • インポートしたファイルやその編集内容がサーバに保存されることはありません。
  • ページをリロードすると、編集内容が消去されます。
